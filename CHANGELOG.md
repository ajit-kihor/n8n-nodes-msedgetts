# Changelog

All notable changes to this project are documented in this file.

## 1.0.0 - 2026-07-20

### Added

- MP3 speech generation with popular and custom Microsoft Edge TTS voices
- Voice listing with normalized names, locales, and genders
- Rate, pitch, and volume controls
- Configurable output binary field and file name
- Multi-item processing with n8n item linking
- Stream timeout, validation, and continue-on-fail handling
- Automated tests and provenance-based npm publishing
